# Phoenix A

Phoenix A is an ongoing research project exploring stateless artificial intelligence architectures governed by explicit invariants rather than adaptive memory.

## Research Areas

- Stateless AI architectures
- Continuity geometry
- Human-directed governance
- Distributed optimization
- Constraint-based execution

## Current Status

Experimental research.

No production implementation is included in this repository.
